Maintenance Request MethodMétodo de Solicitud de Mantenimiento

All requests must be made through the Online Portal by AppFolio.

Request must include:

  • Exact location of the issue
  • Detailed description of the problem

Examples:

  • "In the master bedroom, left corner, there's a switch that doesn't work."
  • "In the second bathroom, the mirror light won't turn on."

Each request should address only one issue.

Response time: up to 24 hours, depending on priority and availability.

Emergency cases are handled urgently.

If there are pets, they must be in a safe area during service.
